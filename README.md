## ai racing sim

tiny pygame racing sim with a human drive mode and a live neuroevolution mode (uses neat-python)

# setup:

    ```bash
    python -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    ```

# run:

    ```bash
    python main.py
    ```
    
# controls:

- `up` accelerate
- `down` brake / reverse
- `left` steer left
- `right` steer right
